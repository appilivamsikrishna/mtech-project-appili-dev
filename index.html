Software Aspects of the Project

This MTech project integrates several advanced software and computational techniques to ensure precise guidance and control in defense applications:

  • Object Tracking and Vision Systems: Using OpenCV, the system detects and tracks objects in real-time, employing algorithms like CSRT to guide the missile's trajectory.
  • Machine Learning and Deep Learning: Incorporates both machine learning and deep learning models for trajectory prediction and correction, refining the missile’s path with live data to ensure accuracy.
  • Data Processing and Algorithms: Utilizes data processing techniques to analyze real-time inputs for trajectory adjustment, enhancing control through deep neural networks.
  • Hardware-Software Integration: Integrates sensors, FPGA, and servo control systems, coordinated by software to ensure stability, orientation, and guidance during operation.
